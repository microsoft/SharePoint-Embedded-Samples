# Tools

Utilities and API clients for working with SharePoint Embedded.

| Tool | Description |
|------|-------------|
| [api-clients](./api-clients) | Postman and Bruno collections for testing Graph API endpoints |
| [migrate-from-blob-storage](./migrate-from-blob-storage) | Console app to migrate files from Azure Blob Storage to SharePoint Embedded |
| [powershell](./powershell) | Admin scripts for container type registration and provisioning |
