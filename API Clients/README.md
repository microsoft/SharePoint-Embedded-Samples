# API Clients for SharePoint Embedded

This folder contains pre-built API client collections for testing and exploring SharePoint Embedded APIs. These collections provide ready-to-use requests for interacting with [Microsoft Graph](https://docs.microsoft.com/graph/overview) and other endpoints used by SharePoint Embedded.

## Available Collections

| Client | Description |
|--------|-------------|
| [Bruno](./Bruno) | A Bruno collection for testing SharePoint Embedded APIs. Requires Bruno v2.1.0 or higher. |
| [Postman](./Postman) | A Postman collection and environment template for testing SharePoint Embedded APIs. |

## Getting Started

1. Choose the API client that best fits your workflow.
2. Follow the setup instructions in the corresponding subfolder's README.
3. Configure the environment variables with your tenant and application-specific information.
